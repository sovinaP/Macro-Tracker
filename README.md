# Macro Tracker Alpha

My first development project so claude played a big part into getting this idea into code

A simple self-hosted calorie and macro tracker that runs in the browser.

## Open the app

[Launch Macro Tracker](https://sovinap.github.io/Macro-Tracker/)

## Features
- Daily calorie and macro tracking (protein, carbs, fat, fiber)
- Quick food buttons for common meals
- Custom food entry
- Body weight log
- 7-day calorie chart
- Dark mode
- Export and import your data as JSON

## Data storage
All data is stored locally in your browser. Use the Export button regularly to back up your data.
